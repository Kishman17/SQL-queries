# sql-analytics-portfolio
Этот репозиторий содержит анонимизированные примеры SQL-запросов, которые я разрабатывал для построения отчетов и анализа данных в реальных проектах. Все запросы написаны для MS SQL и были частью рабочего процесса.

Продемонстрировать навыки:
*   Написания сложных аналитических запросов (оконные функции, агрегация, JOIN).
*   Проектирования структур данных для отчетов.
*   Оптимизации запросов для работы с большими объемами данных.
*   Документирования и оформления кода.
